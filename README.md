## Common Git Commands

```bash
# on main branch
git pull --rebase

# create a new branch
git checkout -b vatsa/question-divide-max-count

# write your code

# add files (. adds all files from the current dir) 
git add .

# commit files
git commit -m "[divide-max] test 1 created"

# push
git push -f

# max more changes

# add again
git add .

# if you want to commit on same commit message
git commit --amend

:wq ENTER

# if you want to commit new 
git commit -m "new commit message"

# again push
git push -f

# review & merge on github

# checkout main
git checkout main

# pull again
git pull --rebase

# delete branch
git branch -D vatsa/question-divide-max-count.

```
